Blog 在 hexo deploy 時發生錯誤

發表於 2018-05-06 |

剛剛要發新文章的時候，
執行 hexo deploy ，突然就拋出以下 Exception

錯誤發生到解決

直接到 _config.yml 檔案裡

原本的設定:

deploy:
type: git
repo: https://github.com/{yourname}/{yourname}.github.io.git
branch: master

修改後的設定:

deploy:
type: git
repo: https://{yourname}:{yourpassword}@github.com/{yourname}/{yourname}.github.io.git
branch: master

參考連結

https://www.zhihu.com/question/38219432
