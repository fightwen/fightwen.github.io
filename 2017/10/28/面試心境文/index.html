面試心境文

Oct 28, 2017 | Hits

突然的想發心境文，因為這些事情，是我自己的事情，
我以為可以跟別人聊出什麼結果，但我並未有獲得，其他人的立足點跟我不同，
或許有的人想這麼在意工作幹嘛，但工作八小時這麼久，如果不是自己在意的事情，根本做不下去。
因為我在意，非常在意。最近沒有一刻不想工作的事情。
想到開始懷疑人生。(ﾟ∀。)
每間公司都寫他們很好玩，哪裡好玩，每間都在快速成長，真的成長了甚麼東西又看不出來。
真的有做出成績的公司真的只有幾間。寥寥可數。

幾次面試下來，有三間工程師覺得我不會去，甚至現在的公司，還被問你怎麼還在這裡，
有時候我想反問，不然我該在哪裡? 為何要一直覺得自己的公司很賽，公司一定有想留的價值阿，不然怎麼會留。
說真的我也不知道我會不會去新公司，
每次拿到offer的時候，我都做了一番很久的掙扎，最後就抽籤決定而已。(好隨便喔，好孩子別學)
因為好幾次面到的都是產品真的滿有興趣的，不然我幹嘛還花時間去他們公司，
但是就是公司內部文化有點奇妙，其實我也不知道自己真正要的是甚麼，一路上都是用刪去法，
用我不要什麼來決定我要什麼，有的工程師是說公司有咖啡機就好了XD，我不需要。
有的工程師希望在家工作，我也不需要。
公司G真的面試面到我很感動，我一直試圖釐清為何那天我很感動。

  • 論福利，光是加班費沒有，全薪病假，那些都輸我現在公司。說起來很弔詭，
    很多公司都期許求職要很有興趣，有興趣到可以超時工作，卻都沒有保證員工超時工作可以獲得相對的休息。
    但這間就是保證加班補休，所以我覺得夠了。
    有的新創也是一進去就給很多特休，但我覺得不是必要。
  • 論人手，也是輸，什麼PM也是RD兼，
    因為他們只配一個人做Android，可是我不需要別人，因為我自己知道我可以做到更好。
  • 論距離，也是輸，離我家真的很遠，但是我覺得我有辦法到公司就好。
  • 論賺錢，一定輸，對方坦白說離開的人覺得沒前景。我覺得前不前景的根本是未知數，當下才是真的。
  • 論技術債，因為我問很多，我知道再怎麼賽，也沒有我公司賽。賽到每個功能都要打掉重練。技術難度我自己可以克服。
    第一年出來面試的時候，我就知道沒有我看不懂的android app code 。
    (這句我覺得太囂張所以保留在我心底，如果以整個Android系統來說，我是謙卑的，因為很大很深)
  • 論自動化，他們沒做，但我覺得進去可以做。
  • 論設備，這真的是贏到沒話說，很少有公司會主動問我需要什麼設備(我都預想好如果公司用win系列一定要問可不可以轉ubuntu)
    對方一直強力推薦要配雙螢幕給我XDD
  • 論產品，本來覺得架上評分不太優，感覺沒甚麼使用者，但真的是面試過才知道，這東西本來通路不在此。
    但都不是重點，我感受到對方很投入自己的產品，不然不會說這麼多。
    有的公司會問用過我們產品嗎，一方面我知道是問求職者有沒有興趣，
    夠不夠了解，但求職者真的只能從使用者的角度去看阿!!
    但是我反問對方的時候，反倒是對方講的又很簡略，有種，阿就做了的口氣，
    有時候都覺得你都不夠了解自己做的產品，不投入這東西，求職者更難投入吧，呵呵。

簡而言之，輸了一大堆，結論，
我只是想找個會講人話 Leader ，公司再多的缺點，
Leader一定是個正常人。可以的話我希望跟我的 Leader 把話說開。
原來那天我就是跟他把話說開，我才這麼想去。

  • 一定要比我還了解產品，知道產品怎麼前進的大方向，連方向都不知道要工程師通靈，不然我知道我就去開公司啦。
  • 樂於分享但他不一定是強到不行，至少我敢跟他表達我的想法，可能當時我的想法很笨，他可以好好回饋我，他不會跟我吵架就好。
  • 給予尊重一定不能用其他Framework去想Android。
  • 不能無限上綱的要員工付出，Leader 卻沒辦法給予相應的回饋。再怎麼投入的產品，最後都會變得沒愛。
  • 不要說謊，不要睜眼說瞎話。缺點可以直白說。反正有問題要敢提敢要求。
  • 他非常加分的地方是，他給予我生存意義這件事，我一直都知道自己心理素質不夠。但我知道別人沒有義務給予我這個。

果然人格特質對我來說比什麼都重要。其次才是追求進步跟很有興趣這件事，追求進步這件事其實跟公司不是一定正相關。
然後我就找不到工作了………..(ﾟ∀。)
還是停下來吧，看幾本書，看幾個有趣開源，都還比較開心。(ﾟ∀。)
